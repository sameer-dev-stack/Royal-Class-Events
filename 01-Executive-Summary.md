## üèóÔ∏è Technical Foundation

### Frontend
- **Framework:** Next.js 14 (App Router) - Server-side rendering, optimal performance
- **Styling:** Tailwind CSS with custom "Royal" theme (Gold & Deep Black)
- **UI Components:** Shadcn UI - Accessible, customizable component library
- **State Management:** React Context + Convex real-time subscriptions
- **Forms:** React Hook Form + Zod validation

### Backend
- **Database:** Convex - Real-time, serverless, type-safe
- **File Storage:** Convex File Storage + CDN integration
- **API Layer:** Convex Functions (queries, mutations, actions)
- **Real-time:** Built-in Convex subscriptions

### AI & Intelligence
- **Primary:** Google Gemini API (event generation, descriptions)
- **Secondary:** OpenAI GPT-4 (advanced reasoning, analytics)
- **Pricing:** Custom ML models (TensorFlow.js for client-side)
- **Demand Prediction:** Time-series analysis (Python microservice)

### Authentication & Payments
- **Auth:** Clerk - Multi-tenant, social logins, RBAC
- **Payments:** Stripe + Local payment gateways (SSLCommerz for BD)
- **Escrow:** Stripe Connect for multi-party settlements

### Infrastructure
- **Hosting:** Vercel (frontend) + Convex Cloud (backend)
- **CDN:** Vercel Edge Network + Cloudflare
- **Monitoring:** Sentry (errors) + Vercel Analytics
- **Email:** Resend (transactional) + SendGrid (marketing)
